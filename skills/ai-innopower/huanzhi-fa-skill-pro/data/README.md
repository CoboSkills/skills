# data/ 目录说明 V1.1.0

## 用途

此目录用于存储 Fa.Pro 用户的数据，包括用户画像、会话记录、上传的文件和生成的简报。

## 目录结构

```
data/
└── users/
    └── [user_id]/
        ├── profile.json          # 用户画像配置
        ├── sessions/             # 会话记录
        │   └── YYYY-MM-DD_session_001.json
        ├── files/                # 上传的文件
        │   └── BP_xxx.pdf
        └── briefs/               # 生成的简报
            └── YYYY-MM-DD_brief.md
```

## 使用说明

1. **用户 ID 生成**: 使用 `user_YYYYMMDD_XXX` 格式
2. **文件命名**: 使用清晰的命名规范，便于检索
3. **数据更新**: 每次会话后及时更新相关文件
4. **隐私保护**: 敏感信息需要加密或脱敏处理

## 保密声明

⚠️ **重要**: 本目录存储的文件默认为可公开版本。用户上传的 BP 等文件应在上传前移除敏感信息。

如需深度保密存储，请联系微信 **Aiinnopower** 升级 Pro 版，使用私有化部署。
