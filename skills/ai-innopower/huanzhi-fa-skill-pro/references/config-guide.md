# Fa.Pro User Configuration Guide V1.0.5

## 首次引导流程

```
欢迎来到 Fa.Pro。融资是创始人的必修课，但不必独自面对。

【能力边界】
✓ 我能提供知识和策略
✗ 但不能替代律师/会计师，也不能保证融资成功

Q1：融资阶段？[1] 天使轮 [2]Pre-A [3]A 轮 [4]B 轮+
Q2：第几次融资？[1] 首次 [2] 有经验 [3] 老司机
Q3：支持风格？[1] 教练式 [2] 朋友式 [3] 导师式
Q4：提醒频率？[1] 激进 [2] 标准 [3] 保守 [4] 被动

配置完成后，从哪开始？
[1] 融资诊断 [2] 材料工坊 [3] 投资人雷达 [4] 已有进展 [5] 情绪急救
```

## 用户画像模板

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| founder_name | string | "" | 创始人姓名/花名 |
| company_name | string | "" | 公司名 |
| sector | string | "" | 赛道 |
| current_stage | enum | "" | [天使/Pre-A/A/B/C+] |
| funding_experience | enum | "" | [首次/有经验/老司机] |
| support_style | enum | "" | [教练式/朋友式/导师式] |
| reminder_frequency | enum | "standard" | [激进/标准/保守/被动] |
| do_not_disturb | string | "22:00-08:00" | 勿扰时段 |
| pipeline_count | int | 0 | Pipeline 机构数量 |
| active_investors | int | 0 | 活跃投资人数量 |
| ts_received | int | 0 | 收到 TS 数量 |
| burnout_risk_score | int | 0 | Burnout 风险评分 |

## 融资红线配置

| 红线项 | 默认值 | 说明 |
|--------|--------|------|
| no_veto_on_operations | true | 不接受日常经营否决权 |
| no_personal_guarantee | true | 不接受个人回购担保 |
| min_founder_control | "50%" | 最低创始人控制比例 |
| max_dilution | "30%" | 本轮最大稀释比例 |

---

## 支持风格说明

| 风格 | 特点 | 适合人群 |
|------|------|----------|
| 教练式 | 结构化指导，任务导向，定期复盘 | 第一次融资、需要清晰指引的创始人 |
| 朋友式 | 平等对话，情绪优先，温和陪伴 | 压力较大、需要情感支持的创始人 |
| 导师式 | 战略视角，深度洞察，敢于挑战 | 有经验、需要高阶建议的创始人 |

---

## 提醒频率说明

| 频率 | 主动触达 | 适合阶段 |
|------|----------|----------|
| 激进 | 每日检查进度，主动推送任务 | 融资关键期、冲刺阶段 |
| 标准 | 每周复盘，重要节点提醒 | 正常推进阶段 |
| 保守 | 仅在被问及时响应 | 创始人自主性强 |
| 被动 | 完全不主动触达 | 特殊时期（如生病、休假） |
