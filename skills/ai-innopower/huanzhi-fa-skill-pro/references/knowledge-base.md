# Fa.Pro Knowledge Base V1.0.5

## 3.1 法规政策库（含 2024 新公司法更新）

### 公司法

| 主题 | 关键要点 | 红线/风险 |
|------|----------|-----------|
| **股权结构** | 创始人应保留控制权至 B 轮前（>50%）；期权池 10-15%，**投前设立**稀释创始人；代持股权需清理 | 股权过于分散；期权池时机不当；联合创始人分配争议 |
| **股东权利** | 普通股 vs 优先股权利差异；保护性条款限于重大决策；否决权不含日常经营 | 小股东日常经营否决；未经同意的强制出售权；过于宽泛的信息披露 |
| **2024 新公司法 - 注册资本** | 有限责任公司股东认缴出资额应在**5 年内缴足**；存量公司有过渡期 | 虚高注册资本风险；需评估实际出资能力 |
| **2024 新公司法 - 股东失权** | 股东未按期足额缴纳出资的，公司可以**失权处理** | 联合创始人出资违约风险 |
| **2024 新公司法 - 董监高责任** | 强化董监高忠实义务和勤勉义务；明确赔偿责任 | 创始人需注意合规履职 |
| **外商投资** | VIE 架构适用于外资限制行业；需 WFOE-OPCO 协议控制；37 号文登记必须 | 政策不确定性；利润转移税务风险；外汇管制资金出境限制 |
| **股权激励** | 期权池占完全稀释后 10-15%；Vesting 通常 4 年，1 年 cliff；**2024 新规：股权激励计划需经股东会决议** | 投前 vs 投后设立影响稀释；行权价格和税务处理 |
| **证券法** | 私募融资合格投资人认定；募资人数限制；信息披露义务 | 非法集资风险；公开宣传违规 |

### 期权池设立（V1.0.5 修正）

**正确表述**：
- 期权池通常在**投前设立**，由创始人股权稀释
- 占**完全稀释后**总股本的 10-15%
- 例如：投前估值 9000 万，期权池 10%，投资 1000 万
  - 投后估值 = 9000 万 + 1000 万 = 1 亿
  - 期权池 = 1 亿 × 10% = 1000 万（由创始人稀释）

**常见陷阱**：
- 投资人要求投后设立期权池 → 实际由投资人共同稀释，创始人吃亏
- 期权池比例过高（>20%）→ 创始人股权过度稀释

### 机构黑话解读

| 投资人说的 | 实际意思 | 应对策略 |
|-----------|----------|----------|
| "我们再看看" | 目前没兴趣，但别得罪 | 定期 Update，等待窗口 |
| "赛道不错，时机早了点" | 不投，但不想明说 | 追问具体顾虑，或放弃 |
| "再看看数据" | 感兴趣但有顾虑 | 主动提供补充材料 |
| "条款都是标准的" | 可能有坑，希望你别细看 | 逐条审阅，找律师 |
| "我们对估值敏感" | 觉得贵，想压价 | 展示可比交易，制造竞争 |
| "在走内部流程" | 可能是真流程，也可能是 Plan B | 设定 Deadline，启动 Plan B |
| "保持联系" | 礼貌性拒绝 | 降级为低优先级，专注其他 |
