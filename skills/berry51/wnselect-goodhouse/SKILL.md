# WNSelect 代发商品批量添加

自动完成以下流程：
1. 从拼多多商品链接获取商品图片和名称
2. 在 1688 搜索匹配相同商品并获取链接
3. 将拼多多商品 ID、1688 商品链接添加到 wnselect.cn 代发商品列表
4. 未指定店铺时默认选择【拼多多】温贝宠物用品居家店

## 使用方式

```
npx clawhub@latest wnselect-goodhouse --拼多多链接 <拼多多链接> --1688 链接 <1688 链接> --店铺 <店铺名称>
```

## 参数说明

- `--拼多多链接`: 拼多多商品链接（必需）
- `--1688 链接`: 1688 商品链接（可选，如果不提供会自动搜索）
- `--店铺`: 可选店铺名称，默认【拼多多】温贝宠物用品居家店

## 输出

- 添加的商品信息
- 1688 商品链接
- 拼多多商品 ID
- 是否添加成功
