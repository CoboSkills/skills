# 亚马逊市场可行性分析

## 核心目标

判断选品方向是否值得进入，输出可执行的选品决策报告。

## 必输参数

| 参数 | 说明 | 必须提供 |
|------|------|---------|
| 品类/关键词 | 分析的目标品类 | ✅ |
| 售价 | 美金 | ✅ |
| 利润率 | 毛利率% | ✅ |
| 数据文件 | SIF导出的xlsx | 推荐上传 |

## 数据来源

**SIF（卖家精灵/Sorftime）** 导出的 xlsx 文件：

| 数据类型 | 分析用途 |
|---------|---------|
| 市场分析 | 市场大盘、品牌格局 |
| 价格带分析 | 找机会价格带 |
| 关键词反查 | 关键词热度分析 |

**注意**：如数据缺失，标注"数据不足"而非编造。

## 分析流程（7步核心框架）

### Step 1：市场大盘分析

- 年/总销量、总销售额
- 近3年同比增幅趋势
- 品牌集中度（CR3/CR5）
- 市场分型（橄榄型/金字塔/扁平型）

### Step 2：品牌格局分析

- Top12品牌销量占比
- 头部品牌特征
- 新品牌进入空间判断

### Step 3：价格带分析（核心！）

7个维度分析每个价格区间：
1. 链接数量 vs 销量分布
2. 平均/最高销量
3. 新品表现
4. 评论数量
5. 评分值
6. 品牌垄断度（CR3/CR5）
7. 卖家归属地（中国/美国/AMZ自营）

### Step 4：目标价格带确定

评分矩阵法：

| 评估维度 | 权重 |
|---------|------|
| 总销量容量 | 25% |
| 新品友好度 | 20% |
| 品牌垄断度 | 20% |
| 中国卖家内卷度 | 15% |
| 评论门槛 | 10% |
| 利润空间 | 10% |

### Step 5：产品属性分析

- 退货率及Top5退货原因
- 加权平均点击转化率
- 颜色/尺码销量分布

### Step 6：利润空间测算

```
纯利润 = 售价 - 采购成本 - 头程物流 - FBA配送费 - 佣金(17%) 
         - 配置费 - 退货成本(售价×退货率) - 广告费(售价×广告占比)

利润率 = 纯利润 ÷ 售价 × 100%
```

支持空运/海运两种场景。

### Step 7：风险矩阵评估

| 风险类型 | 等级 | 应对策略 |
|---------|------|---------|
| 竞争风险 | 高/中/低 | 价格战预案 |
| 季节性风险 | 高/中/低 | 库存管理 |
| 品质风险 | 高/中/低 | 品控方案 |
| 资金风险 | 高/中/低 | 现金流规划 |

## 输出格式

### xlsx报告（6 sheets）

1. **市场大盘** - 规模/趋势/品牌集中度
2. **价格带分析** - 7维度详细数据
3. **目标价格带推荐** - 评分矩阵 + 推荐区间
4. **产品属性** - 退货率/转化率/颜色尺码
5. **利润测算** - 空运/海运两档
6. **选品决策** - 核心结论 + 风险提示 + 行动清单

### 对话总结模板

```
## 市场概况
- 市场规模：年销XX万件，销售额$XX
- 市场类型：XX市场，近3年XX
- 品牌集中度：CR3=XX%，CR5=XX%

## 推荐价格带
**[$XX-$XX]**（综合评分X.XX分）

## 核心风险
1. 退货率XX%，主要来自XX/XX问题
2. 中国卖家占比XX%，存在XX风险
3. 季节性明显，需精准库存管理

## 利润测算
| 场景 | 利润率 | 最低盈亏定价 |
|------|--------|------------|
| 海运 | XX% | $XX |
| 空运 | XX% | $XX |

## 行动建议
1. 【产品】优先解决XX问题
2. 【定价】建议$XX-$XX
3. 【备货】首批XX件
4. 【上架】最晚XX月前完成

## 选品决策
**[Go / No-Go / 谨慎进入]**
```

## 注意事项

- 所有数值计算必须用Python完成
- 百分比统一保留2位小数
- 金额统一用美金，保留2位小数
- 结论必须有数据支撑
- 数据缺失时标注"数据不足，建议补充XX数据"
