# 高考辅助助手

## 功能概览

本技能提供高考全流程的辅助支持，覆盖志愿填报、院校分析、政策解读、考前准备等关键环节。

## 核心功能

### 1. 志愿填报

使用以下工具进行志愿分析：

- **院校/专业查询** - 查询各大院校基本信息、专业设置、录取分数
- **录取概率计算** - 根据分数、位次、选科计算录取概率
- **冲稳保策略** - 制定三梯度志愿组合方案
- **政策匹配** - 识别适用的特殊政策和照顾政策
- **当年分数查询** - 根据高考分数和位次，查询可投档院校和专业（**新增核心功能**）

相关参考文件：
- [volunteer-policy.md](references/volunteer-policy.md) - 各省志愿填报规则和录取政策
- [university-data.md](references/university-data.md) - 院校和专业数据库
- [score-query.md](references/score-query.md) - 当年高考分数可投档院校专业查询（**新增核心功能**）

### 2. 院校专业特点

分析院校和专业信息：

- **院校特色** - 985/211/双一流、学科优势、地理位置
- **专业详解** - 课程设置、就业前景、发展方向
- **对比分析** - 多维度对比不同院校和专业

相关参考文件：
- [university-analysis.md](references/university-analysis.md) - 院校专业深度分析指南

### 3. 招生政策

了解最新招生政策：

- **国家/省政策** - 特殊类型招生、专项计划、加分政策
- **高校政策** - 强基计划、综合评价、艺术类招生
- **照顾政策** - 少数民族、退役士兵、山区县等优惠政策

相关参考文件：
- [admissions-policies.md](references/admissions-policies.md) - 招生政策详解

### 4. 高考前注意事项

考前准备指导：

- **物资准备** - 证件、文具、生活用品清单
- **行程规划** - 考点定位、交通安排、应急预案
- **心态调整** - 压力管理、睡眠调节、考场技巧
- **心理健康** - 临场焦虑应对、失眠紧急处理、心态维护

相关参考文件：
- [exam-prep.md](references/exam-prep.md) - 考前准备指南
- [mental-health.md](references/mental-health.md) - 心理健康与焦虑应对指南（**新增**）

### 5. 女生考前特别关怀

针对女生群体的特殊建议：

- **生理期管理** - 提前规划、饮食调理、应急准备
- **生活建议** - 住宿安排、安全防护、心理支持
- **营养健康** - 特殊营养需求、避免的食物

相关参考文件：
- [girl-specific.md](references/girl-specific.md) - 女生高考特别指南

## 使用流程

1. **明确需求** - 说明当前阶段（备考/填报志愿/考前准备）
2. **提供信息** - 分数、位次、选科、省份、意向方向
3. **获取方案** - 获取个性化建议和具体操作步骤
4. **追问细化** - 针对具体问题进一步咨询

## 注意事项

- 招生政策会随年份变化，需参考最新官方信息
- 志愿填报需结合个人兴趣和职业规划
- 女生生理期等特殊情况需提前规划和准备
- 心理健康问题严重时应及时寻求专业帮助

**重要提醒**：本技能提供的是辅助参考，最终决策请结合官方信息和个人实际情况。
