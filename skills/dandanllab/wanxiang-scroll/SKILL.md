# 万象绘卷

> 全能故事创作与交互体验工具箱 | v2.6.8

## 功能概览

- **56种文风切换**：从殿堂文学到网文爽文，一键切换
- **网文创作全流程**：黄金三章、细纲生成、平台适配
- **交互式故事模拟**：实时互动、角色扮演、后台成书
- **异世界人生模拟**：回合制人生体验、世界观随机生成、代代相传、自由行动AP系统、后台成书
- **剧情管理系统**：长篇故事管理、大总结系统
- **拆书融合创作**：多源小说融合创作新作品
- **小说档案馆**：爬取、分析、存储小说素材

## 四种工作模式

### 1. 创作学习模式（默认）
使用 `#万象` 指令进入创作模式，提供：
- 文风切换与效果测试
- 网文创作全流程指导
- 拆书分析与融合创作
- 小说档案馆素材管理

### 2. 交互式故事模式
使用 `#主菜单` 指令进入游戏模式，体验：
- 实时互动角色扮演
- 剧情分支与选择
- 后台自动成书
- 大总结系统

### 3. 异世界人生模拟模式
使用 `#人生模拟` 指令进入，体验：
- 回合制人生体验
- 世界观随机生成
- 代代相传系统
- 自由行动AP系统

### 4. 角色扮演模式
使用 `#角色扮演` 指令进入，提供：
- 沉浸式角色扮演
- 场景化互动体验
- 后台成书记录

## 快速开始

### 文风切换示例

```
用户：#切换文风 网文爽文
AI：[切换到网文爽文风格]
    特征：快节奏、强冲突、爽点密集
    语气：直白有力、情绪饱满
    句式：短句为主、段落简洁

用户：写一段主角逆袭的场景
AI：[以网文爽文风格输出...]
```

### 网文创作示例

```
用户：#万象
AI：🎨 万象绘卷 - 创作模式
    请选择功能：
    1. 黄金三章生成
    2. 细纲设计
    3. 文风切换
    4. 平台适配
    5. 拆书分析

用户：1
AI：请输入小说题材和核心设定...
```

- Python 3.10+
- 磁盘空间 100MB+

```bash
pip install -r requirements.txt
python scripts/init_database.py
```
