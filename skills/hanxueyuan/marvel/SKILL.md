# Marvel (漫威娱乐)

美国著名漫画出版商，隶属于迪士尼。

## 公司介绍

**漫威娱乐**成立于 1939 年，2009 年被迪士尼收购，拥有 8000+ 个漫画角色。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1939 年 |
| **总部** | 美国纽约 |
| **母公司** | 迪士尼 |
| **竞争对手** | DC 漫画 |
