# Nintendo 任天堂技能

## 技能描述
Nintendo（任天堂）是日本知名游戏公司，全球领先的游戏机、游戏软件制造商，以 Switch、Mario、Zelda 等闻名。本技能帮助用户查询 Nintendo 产品、游戏信息。

## 核心功能
- 🎮 **产品查询**：Switch、游戏卡带、配件
- 🎯 **游戏查询**：第一方游戏、第三方游戏
- 📅 **新品发布**：最新游戏、主机资讯
- 💰 **价格查询**：产品价格范围
- 🌟 **经典 IP**：Mario、Zelda、Pokemon、Splatoon 等
- 🏪 **门店查询**：任天堂直营店、授权经销商

## 使用场景
- 查询 Nintendo 最新产品
- 了解游戏信息
- 查找附近门店
- 获取游戏推荐
- 了解经典 IP

## 注意事项
- Nintendo 是日本游戏公司
- 成立于 1889 年，总部位于京都
- Switch 是当前主力游戏机
