# 长江存储 YMTC 技能

## 技能描述
长江存储（Yangtze Memory Technologies Corp，简称 YMTC）是中国领先的 NAND Flash 存储器制造商，总部位于武汉。本技能帮助用户查询长江存储产品、技术信息。

## 核心功能
- 💾 **产品查询**：3D NAND、SSD、嵌入式存储
- 🔧 **技术支持**：产品规格、技术方案、开发工具
- 📊 **公司信息**：企业简介、发展历程、技术路线
- 🏪 **销售渠道**：授权代理商、购买渠道
- 📚 **开发资源**：数据手册、应用指南、参考设计
- 🎓 **行业资讯**：存储市场动态、技术趋势

## 主要产品线
- **3D NAND Flash**：Xtacking 架构闪存芯片
- **消费级 SSD**：致钛（TiPlus）系列
- **企业级 SSD**：数据中心存储解决方案
- **嵌入式存储**：UFS、eMMC 产品

## 核心技术
- **Xtacking 架构**：创新的 3D NAND 技术
- **堆叠层数**：128 层、232 层 3D NAND
- **接口标准**：PCIe、NVMe、SATA

## 使用场景
- 查询 NAND Flash 产品规格
- 获取 SSD 选型建议
- 了解存储技术方案
- 查找授权代理商
- 获取行业资讯

## 注意事项
- 长江存储是中国最大的 NAND Flash 制造商
- 致钛是长江存储消费类品牌
- 建议通过官方渠道购买产品
