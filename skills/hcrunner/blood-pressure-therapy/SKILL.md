# 高血压治疗管理 Skill

## 描述
本技能提供高血压治疗的完整管理流程，帮助用户进行基础数据采集、日常血压监测、AHA 2017标准分类评估、危险预警和音频治疗。音频经过专业模型训练，可根据用户性状和血压情况，给出个性化的放松音频方案。

## 激活条件
当用户提到以下关键词或场景时激活此技能：
- 高血压、血压高、血压问题、血压异常
- 需要监测血压、血压管理、记录血压
- 血压控制、降压
- 收缩压、舒张压、脉压差
- 血压趋势分析、血压预警
- 高血压音频治疗、放松音乐
- 录入血压、测量血压

### Step 3 & 4: 血压分类评估 + 音频治疗方案（⚠️ 必须同时输出）

#### 3.1 血压分类判定（参考 [references/AHA_2017.md](references/AHA_2017.md)）

根据 AHA / ACC 2017 标准：

| 分类 | 收缩压 (mmHg) | 舒张压 (mmHg) | 说明 |
|------|:---:|:---:|------|
| **正常 (Normal)** | < 120 | **且** < 80 | 理想状态 |
| **升高 (Elevated)** | 120–129 | **且** < 80 | 前期预警，需关注 |
| **1级高血压** | 130–139 | **或** 80–89 | 需要干预 |
| **2级高血压** | ≥ 140 | **或** ≥ 90 | 需积极治疗 |
| **高血压急症** | > 180 | **和/或** > 120 | 可能需紧急就医 |

#### 3.2 各分类对应建议 & 3.3 脉压差计算 → [references/classification_advice.md](references/classification_advice.md)

#### 4. 音频治疗方案（参考 [assets/blood_pressure_therapy.md](assets/blood_pressure_therapy.md)）

根据 Step 1 采集的用户性别匹配音频资源，同一次回复一并输出。

**音频介绍：**
这是一套专为高血压人群设计的放松音频方案，通过特定的频率和节奏引导身心进入深度放松状态，帮助降低交感神经活性，从而对血压产生积极的辅助调节作用。结合深呼吸练习效果更佳。

**音频使用建议（每次都附带）：**
- 每天 1–2 次，每次 15–30 分钟
- 在安静环境中使用
- 音量控制在 30–50%
- 可与深呼吸练习结合效果更佳
- 最佳使用时机：睡前或血压升高感到不适时
- 音频治疗是辅助放松手段，不能替代药物治疗

## 重要说明

- 本技能仅供辅助健康管理，**不替代专业医疗诊断和建议**
- 高血压急症（>180/120）尤其伴症状时，务必立即就医
- 用药调整必须在医生指导下进行
- 音频治疗为辅助放松手段，不能替代药物治疗
- 建议定期体检，长期跟踪血压控制效果
