# 百度智能云 VOD 字幕擦除 Skill

智能识别并擦除视频中的字幕区域，支持多种擦除模式和高级配置。

## ⚠️ 强制执行流程

**禁止直接执行命令！必须按以下流程引导用户：**

```
1. 识别意图 → 判断擦除类型（字幕/Logo/时间范围）
2. 收集参数 → 使用 Question 工具询问用户
3. 确认配置 → 展示参数让用户确认
4. 用户确认 → 才能执行命令
```

**错误示例（禁止）：**
```
用户: 帮我擦除这个视频的字幕
Agent: [直接执行 python3 desubtitle.py video.mp4 --mode global]  ❌
```

**正确示例：**
```
用户: 帮我擦除这个视频的字幕
Agent: 请问视频是什么类型？
  [1] 电影/电视剧（底部对白字幕）
  [2] 综艺/广告（可能有多处字幕）
用户: 1
Agent: 请提供视频文件路径：
用户: /Downloads/video.mp4
Agent: [执行命令]  ✓
```

## 触发规则

满足以下任一条件时触发：

1. 用户提及"字幕擦除"、"擦除字幕"、"去除字幕"
2. 用户需要处理视频中的字幕
3. 用户需要查询或删除字幕擦除任务

## Agent 参数收集指南

### 核心原则

1. **场景优先**：先理解用户场景，再推荐参数
2. **最小必要**：只收集必要参数，其他使用默认值
3. **渐进式**：从简单到复杂，避免一次性问太多
4. **推荐优先**：给出推荐选项，而非开放式问题

---

## 第一步：识别用户意图

根据用户描述，识别属于哪种场景：

| 用户描述关键词 | 场景类型 | 推荐模式 |
|---------------|----------|----------|
| "擦除字幕"、"去除字幕"、"消字幕" | 字幕擦除 | 根据视频类型判断 |
| "擦除Logo"、"去除水印"、"去掉角标" | 区域擦除 | manual |
| "只擦除某一段"、"前几分钟" | 时间范围 | global + time-range |
| "查任务"、"任务状态" | 任务查询 | --list/--query |
| "删除任务" | 任务删除 | --delete |

---

## 第二步：收集必要参数

### 场景 A：字幕擦除（最常见）

**必须收集**：
1. 视频来源（本地路径/网盘路径）

**根据视频类型推荐模式**：

| 视频类型 | 推荐模式 | 原因 |
|----------|----------|------|
| 电影、电视剧、访谈 | dialog | 只擦除底部对白字幕 |
| 综艺、广告、宣传片 | global | 可能有多处字幕 |
| 用户明确说"所有字幕" | global | 全局擦除 |
| 用户提到具体位置 | manual | 自定义区域 |

**交互模板**：

```
用户: 帮我把这个视频的字幕擦除

Agent: 好的，请问视频是什么类型？
  [1] 电影/电视剧/访谈（底部对白字幕）
  [2] 综艺/广告/宣传片（可能有多处字幕）
  [3] 其他类型

用户: 1

Agent: [自动选择 dialog 模式]
请提供视频文件路径：
```

### 场景 B：区域擦除（Logo/水印）

**必须收集**：
1. 视频来源
2. 擦除区域坐标

**交互模板**：

```
用户: 帮我去掉视频左上角的Logo

Agent: 好的，我需要知道Logo的位置信息。
请提供Logo区域的坐标（格式: x,y,宽度,高度）：
  - x: Logo左边距离视频左边的像素
  - y: Logo上边距离视频上边的像素
  - 宽度: Logo的宽度
  - 高度: Logo的高度

提示：可以用视频播放器截图，在图片编辑器中查看像素坐标。

用户: 50,30,200,80

Agent: 收到，Logo区域: (50,30) 宽200 高80
是否还有其他需要擦除的区域？

用户: 没有

Agent: [执行: --mode manual --areas "50,30,200,80"]
```

### 场景 C：时间范围擦除

**必须收集**：
1. 视频来源
2. 开始时间
3. 结束时间

**交互模板**：

```
用户: 只擦除视频前30秒的字幕

Agent: 好的，只擦除前30秒。
请提供视频文件路径：

用户: /Downloads/video.mp4

Agent: [执行: --mode global --time-range 0 30000]
```

## 第三步：确认高级参数（可选）

**仅在以下情况询问**：

| 参数 | 何时询问 | 默认值 |
|------|----------|--------|
| 模型类型 | 用户提到"快点"、"赶时间" | v4 |
| 输出目录 | 用户指定保存位置 | ./output |
| 上传网盘 | 视频来自网盘时主动询问 | 否 |

**不要主动询问的参数**：
- model（默认 v4 效果最好）
- areas（manual 模式必须，其他模式不需要）
- time-range（用户不提就不需要）

## 推荐参数组合

### 组合 1：影视剧字幕擦除（推荐）

```bash
python3 scripts/desubtitle.py <视频> --mode dialog
```

**适用场景**：电影、电视剧、访谈节目
**特点**：只擦除底部对白字幕，保留其他文字

### 组合 2：综艺/广告字幕擦除

```bash
python3 scripts/desubtitle.py <视频> --mode global
```

**适用场景**：综艺节目、广告、宣传片
**特点**：智能识别并擦除所有字幕区域

### 组合 3：Logo/水印去除

```bash
python3 scripts/desubtitle.py <视频> --mode manual --areas "x,y,w,h"
```

**适用场景**：去除特定位置的Logo、水印、角标

### 组合 4：部分时间擦除

```bash
python3 scripts/desubtitle.py <视频> --mode global --time-range 0 30000
```

**适用场景**：只擦除视频某一段的字幕

### 组合 5：快速处理

```bash
python3 scripts/desubtitle.py <视频> --mode dialog --model v3
```

**适用场景**：赶时间、视频较长
**特点**：使用 v3 模型，速度快但效果略差

## 常见问题处理

### Q1：用户不知道视频类型

```
Agent: 您的视频是什么内容？
  [1] 电影/电视剧 - 有底部对白字幕
  [2] 综艺节目 - 可能有花字、特效字幕
  [3] 广告/宣传片 - 可能有标题、说明文字
  [4] 不确定 - 使用全局擦除（推荐）

用户: 4

Agent: 好的，使用全局擦除模式，会自动识别所有字幕区域。
```

### Q2：用户不知道坐标

```
用户: 我要擦除Logo，但不知道坐标

Agent: 您可以：
  1. 用视频播放器截图
  2. 在图片编辑器（如PS、画图）中打开截图
  3. 查看Logo左上角的坐标(x,y)和尺寸(宽,高)

或者告诉我Logo的大概位置：
  [1] 左上角
  [2] 右上角
  [3] 左下角
  [4] 右下角

用户: 右上角

Agent: 请告诉我Logo大概占视频宽高的比例？
  例如：宽200像素，高50像素

用户: 宽150，高40

Agent: 好的，右上角Logo，宽150高40。
假设视频是1920x1080，Logo坐标大约是：
  x=1770, y=0, width=150, height=40

确认使用这个坐标吗？如果不确定，建议先截图确认。
```

### Q3：用户要处理多个视频

```
用户: 帮我擦除这三个视频的字幕

Agent: 好的，请提供三个视频的路径：
  1. 第一个视频：
  2. 第二个视频：
  3. 第三个视频：

用户: a.mp4 b.mp4 c.mp4

Agent: 三个视频都使用相同的擦除模式吗？
  [1] 是，都用 dialog（影视剧）
  [2] 是，都用 global（全局）
  [3] 不，每个视频不同

用户: 1

Agent: [执行: a.mp4 b.mp4 c.mp4 --mode dialog]
```

## 执行命令模板

### 基础命令

```bash
export BAIDU_VOD_AK='xxx' && export BAIDU_VOD_SK='xxx' && \
python3 scripts/desubtitle.py <视频路径> --mode <模式>
```

### 完整命令（所有参数）

```bash
export BAIDU_VOD_AK='xxx' && export BAIDU_VOD_SK='xxx' && \
python3 scripts/desubtitle.py <视频路径> \
  --mode <global|dialog|manual> \
  --model <v3|v4> \
  --areas "x,y,w,h" \
  --time-range <开始ms> <结束ms> \
  --output-dir <输出目录> \
  --upload-netdisk \
  --netdisk-dir <网盘目录>
```

## 前置条件

### 1. VOD 凭证

```bash
export BAIDU_VOD_AK="your_access_key"
export BAIDU_VOD_SK="your_secret_key"
```

获取方式：
1. 登录 [百度智能云控制台](https://console.bce.baidu.com/)
2. 开通 [智能点播平台 VOD](https://cloud.baidu.com/product/vod.html)
3. 在 [安全认证](https://console.bce.baidu.com/iam/#/iam/accesslist) 获取 AK/SK

### 2. 网盘功能（可选）

如需从网盘下载或上传到网盘，需要：
1. 安装 bdpan CLI
2. 执行 `bdpan login` 登录

---

## 处理时间

- 约 5-15 分钟（取决于视频长度）
- 可使用 `--no-wait` 参数不等待，稍后查询任务状态

---

## 注意事项

1. **文件大小限制**：单个文件 ≤ 5GB
2. **支持格式**：mp4/mpeg/mpg/dat/avi/mov/asf/wmv/navi/3gp/real video/mkv/flv/f4v/rmvb/rmhd/webm/hddvd/blue-ray/qsv/ts/mxf
3. **费用**：按处理时长计费
4. **白名单**：字幕擦除需要白名单权限，如提示"权限不足"请联系百度智能云开通
