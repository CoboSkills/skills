# 法律文书润色AI助手 ✨

> 专业法律文书润色工具 | 12种润色维度 | 让文书更专业、更有说服力

## 🎯 适用场景

- 起诉状/答辩状润色
- 代理词/辩护词优化
- 合同条款精炼
- 法律意见书提升
- 律师函美化
- 证据目录整理
- 各类法律文书语言优化

## 🎨 12种润色维度

### 1. 观点深度拓展
```
请对以下法律文书进行观点深度拓展：

[粘贴文书内容]

润色要求：
1. 挖掘深层法律问题
2. 补充理论依据
3. 增加学理分析
4. 引用权威观点
5. 提升论证层次

输出：润色后的完整文书
```

### 2. 语言简洁凝练
```
请对以下法律文书进行语言精简：

[粘贴文书内容]

润色要求：
1. 删除冗余表述
2. 合并重复内容
3. 使用法律术语替代口语
4. 每句话都有信息量
5. 控制篇幅

输出：精简后的文书（标注删减字数）
```

### 3. 创新元素注入
```
请对以下法律文书注入创新元素：

[粘贴文书内容]

润色要求：
1. 新颖的论证角度
2. 独特的法律解释
3. 创新的救济方案
4. 前瞻性建议
5. 突破常规思维

输出：创新优化后的文书
```

### 4. 整体和谐把控
```
请对以下法律文书进行整体和谐性优化：

[粘贴文书内容]

润色要求：
1. 前后逻辑一致
2. 各部分比例协调
3. 详略得当
4. 过渡自然
5. 首尾呼应

输出：结构优化后的完整文书
```

### 5. 风格统一协调
```
请对以下法律文书进行风格统一：

[粘贴文书内容]

润色要求：
1. 人称统一（我方/原告/代理人）
2. 时态统一
3. 语气一致（正式/恳切/强硬）
4. 格式规范统一
5. 法律引用格式统一

输出：风格统一后的文书
```

### 6. 文化元素融入
```
请对以下法律文书融入适当的文化元素：

[粘贴文书内容]

润色要求：
1. 引用法律谚语
2. 借鉴经典判词
3. 体现法律文化
4. 增强人文气息
5. 提升说服力

注意：适度使用，不喧宾夺主

输出：文化润色后的文书
```

### 7. 词汇精准替换
```
请对以下法律文书进行词汇精准化：

[粘贴文书内容]

润色要求：
1. 口语→法言法语
2. 模糊词→精确表述
3. 情绪词→中性客观
4. 加强法律术语使用
5. 统一专业表述

示例：
- "很多钱" → "人民币XX万元"
- "不公平" → "显失公平，违反公平原则"
- "欺负" → "利用优势地位"

输出：词汇优化后的文书（附替换对照表）
```

### 8. 句式多样优化
```
请对以下法律文书进行句式优化：

[粘贴文书内容]

润色要求：
1. 长短句结合
2. 避免连续长句
3. 重要内容前置
4. 使用排比增强气势
5. 设问引发思考

输出：句式优化后的文书
```

### 9. 逻辑脉络梳理
```
请对以下法律文书进行逻辑梳理：

[粘贴文书内容]

润色要求：
1. 明确论点-论据-结论

4. 消除逻辑跳跃

输出：逻辑优化后的文书（附逻辑结构图）
```

结尾优化：
1. 总结升华
2. 强化诉求
3. 情感共鸣
4. 有力收尾

输出：开头结尾优化后的完整文书
```

### 11. 情感深度挖掘
```
请对以下法律文书进行情感优化：

[粘贴文书内容]

润色要求：
1. 挖掘案件背后的情感因素
