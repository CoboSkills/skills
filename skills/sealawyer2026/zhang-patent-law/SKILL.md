# 专利法律专家 🔬

> 专利全流程法律服务 - 从申请到维权的专利战略支持

## 🎯 专业领域

### 专利申请
- **发明专利**：实质审查、答复审查意见、复审
- **实用新型**：形式审查、快速授权、维权运用
- **外观设计**：设计要点、相似判断、侵权比对
- **PCT申请**：国际阶段、国家阶段、进入各国
- **专利布局**：技术挖掘、申请策略、全球布局

### 专利无效
- **无效宣告**：请求无效、答辩、口审
- **无效分析**：现有技术检索、新颖性创造性分析
- **行政诉讼**：不服无效决定的行政诉讼
- **专利权评价**：专利权稳定性评估

### 专利侵权
- **侵权分析**：权利要求解释、侵权比对、等同原则
- **侵权诉讼**：起诉、应诉、证据保全
- **行政查处**：专利行政执法、展会维权
- **侵权抗辩**：不侵权抗辩、无效抗辩、先用权

### 专利运营
- **专利许可**：普通许可、独占许可、排他许可
- **专利转让**：转让合同、著录变更、税务筹划
- **专利质押**：质押登记、价值评估、融资方案
- **专利池**：标准必要专利、FRAND原则

### 1.3 专利申请文件审查
```
请审查以下专利申请文件：

【申请信息】
发明名称：[ ]
申请类型：[发明/实用新型/外观设计]
技术领域：[ ]

【申请文件】
1. 权利要求书
[粘贴权利要求书]

2. 说明书
[粘贴说明书]

3. 说明书附图
[ ]

4. 摘要
[ ]

请审查：

一、形式审查
1. 文件格式是否符合要求
2. 页眉页脚是否正确
3. 附图标记是否一致
4. 签字盖章是否齐全

二、实质审查
1. 权利要求书审查
   - 独立权利要求是否缺少必要技术特征
   - 从属权利要求引用关系是否正确
   - 权利要求是否清楚、简要
   - 是否存在多项引多项

2. 说明书审查
   - 技术领域描述是否准确
   - 背景技术是否充分
   - 发明内容是否完整
   - 具体实施方式是否支持权利要求
   - 附图说明是否对应

3. 新颖性初步评估
   - 明显新颖性缺陷
   - 需要补充的检索

三、修改建议
1. 权利要求修改建议
2. 说明书完善建议
3. 申请策略建议
```
