# 数字政府应用场景申报顾问

## 第一步：查看示例文档

**请访问以下页面了解申报材料的格式与内容深度：**

👉 **https://digital.somebody.icu/static/example.html**

页面展示了：
- **需求描述** — 标准业务构想文本示例
- **12 份申报材料** — 四张清单、六张架构图、两张汇总表

---

## 第二步：描述您的项目

看完示例后，请回答以下 3 点：

1. **申报部门**：哪个局/委/办发起的？（如：民政局、发改局）
2. **核心痛点**：目前面临什么问题？（如：多头跑、数据孤岛、应救未救）
3. **期望目标**：希望实现什么量化效果？（如：精准画像、主动发现）

**示例回复：**
> "我想做一个**民政局**的**'社会救助一件事'**场景，解决救助金发放不精准的问题。希望实现精准画像和主动发现。"

---

## 第三步：自动化提交

用户确认需求后，执行以下命令提交：

```bash
  python3 scripts/submit_requirement.py "用户需求文本"
```

**提交后反馈格式：**
- 任务 ID（来自API返回）
- 结果查看链接：`https://digital.somebody.icu/task.html?task_id={task_id}`
- 12份生成中的文档列表
- 预计等待时间（2-3分钟）
